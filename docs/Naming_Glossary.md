# Naming Glossary

## Purpose

This note explains the main names used across the public documentation so readers can distinguish business branding, organization identity, and legacy technical naming.

## Emerald Insurance

Emerald Insurance is the public business-facing brand used for the current proof-of-concept verifier journeys and customer-facing documentation.

Use this name when describing:

- the demo insurer
- the verifier journeys
- the customer and agent-facing proof-of-concept experience

## InsTech.ie

InsTech.ie is the industry-readiness context used in the public org profile and insurer-facing material.

Use this name when describing:

- industry readiness work
- insurer engagement
- future sandbox participation and evaluation context

## InstechSandbox

InstechSandbox is the GitHub organization that hosts the repositories used in this proof of concept.

Use this name when referring to:

- GitHub repositories
- repository ownership
- source and release locations
